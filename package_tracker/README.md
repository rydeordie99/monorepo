Personal package tracker for multiple carriers.
Requires internal django site (code not currently public).

Requires the following environment variables  
INTERNAL_API_URL  
INTERNAL_API_KEY

and the following which can be obtained by registering at the respective shipping company developer sites  
CP_API_KEY  
UPS_CLIENT_ID  
UPS_CLIENT_SECRET  
FEDEX_CLIENT_ID  
FEDEX_CLIENT_SECRET  
PUROLATOR_KEY  
PUROLATOR_PASS  
DHL_API_KEY
